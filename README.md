Репозитий с материалами к воркшопу

Теги в проекте:

- initial - начальный проект
- separate - разделенный логин на компоненты
- hoc - реализация логина на HOCах
- recompose - Логин на HOC'ах + recompose
- renderprops - Логин на render props
- architecture - "правильная" архитектура проекта
- storybook - демонстрация storybook
- badredux - "плохая" реализация на redux
- hyperthunk - "демонстрационная" реализация отмен запроса на thunk
- sagas - реализация "сокета" на сагах
